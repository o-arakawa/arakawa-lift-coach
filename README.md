# Arakawa Lift Coach

筋トレ版・荒川理専用チャッピーのMVPです。

## できること

- いつものメモ感覚で筋トレを記録
- 「ベンチ60×8、65×5」形式から重量・回数・ボリュームをざっくり抽出
- 何日筋トレをしていないかを表示
- 疲労、飲酒、肩痛、股関節痛、目的から今日のメニューを提案
- 部位別ボリュームをグラフ表示

## 使い方

`index.html` をブラウザで開くだけで動きます。

## GitHub Pages

GitHubにアップしたあと、Repository Settings の Pages で `main` ブランチのルートを公開すると、そのままWebアプリとして使えます。

## 次に入れたい機能

- Googleスプレッドシート保存
- Googleカレンダー連携
- OpenAI APIで自然文ログの精度向上
- Firebaseログインとクラウド保存
- LINE入力連携
